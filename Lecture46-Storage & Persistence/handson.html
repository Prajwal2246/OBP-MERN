LocalStorage

  • key value pairs both in strings.
  • origin : domain(www.google.com) and subdomain(www.docs.google.com)
  • for each origin browser allocates 5-10mb space
  • persists -> reload,reopen browser,tab reopen,
  • data deletes when we use removeItem and .clean() method
    Limitaitions
  • large data cannot be store
  • data limited to strings
  • QuotaExceededError
  • setItem(write) is expensive task so can give performance issue
  • security issues (personal / sensitive info)
